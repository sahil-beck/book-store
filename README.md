# Book Store

## Features

- CRUD operations for books
- Responsive frontend
- MongoDB database

## Tech Stack

Frontend
- React
- Vite
- Tailwind CSS

Backend
- Node.js
- Express
- MongoDB
- Mongoose

## Project Structure

backend/
frontend/

## Environment Variables

Create backend/.env

PORT=5555
MONGODB_URL=
FRONTEND_URL=http://localhost:5173

Create frontend/.env
VITE_BACKEND_URL=http://localhost:5555

## Installation

### Backend

cd backend
npm install
npm start

### Frontend

cd frontend
npm install
npm run dev
